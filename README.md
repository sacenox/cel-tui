# cel-tui

A TypeScript TUI framework built around a concise API and ultra-fast rendering.

## Why "cel"?

A **cel** is the smallest unit of a terminal display — a single character cell. It represents the atomic building block of every terminal interface. The name reflects the library's philosophy: start from the smallest meaningful unit and build up, with nothing wasted.

## Goals

- **Concise API** — minimal surface area, maximum expressiveness
- **Ultra fast** — performance-first rendering and state updates
- **TypeScript native** — full type safety out of the box
- **Terminal native** — built for the terminal, not ported from the browser
